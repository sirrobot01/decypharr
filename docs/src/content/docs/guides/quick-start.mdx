On first run, Decypharr presents a setup wizard to configure core settings.

## Step 1: Authentication

<img
	src={authSetupImage.src}
	width={authSetupImage.width}
	height={authSetupImage.height}
	alt="Setup wizard authentication screen"
/>
*Screenshot: Authentication setup screen.*

Create your admin credentials:
- **Username**: Your login username
- **Password**: Secure password

:::tip
The API token is auto-generated and shown after setup completes.
:::

## Step 2: Debrid Providers

<img
	src={debridSetupImage.src}
	width={debridSetupImage.width}
	height={debridSetupImage.height}
	alt="Setup wizard debrid providers screen"
/>
*Screenshot: Debrid provider selection.*

Add at least one Debrid provider:

### Supported Providers
- **Real Debrid** (`realdebrid`)
- **All Debrid** (`alldebrid`)  
- **Debrid Link** (`debrid-link`)
- **Torbox** (`torbox`)
- **Premiumize** (`premiumize`)

For each provider:
1. Select provider type
2. Enter API key (from provider dashboard)
3. *(Optional)* Configure rate limits

## Step 3: Usenet Providers (Optional)

<img
	src={usenetSetupImage.src}
	width={usenetSetupImage.width}
	height={usenetSetupImage.height}
	alt="Setup wizard Usenet configuration screen"
/>
*Screenshot: Usenet provider configuration.*

Add NNTP servers for Usenet support:

**Required:**
- Host (e.g., `news.provider.com`)
- Port (typically `563` for SSL)
- Username
- Password

**Optional:**
- Max connections (default: 20)
- SSL enabled (recommended)
- Priority (for multiple providers)

## Step 4: Mount Configuration

<img
	src={mountSetupImage.src}
	width={mountSetupImage.width}
	height={mountSetupImage.height}
	alt="Setup wizard mount type selection screen"
/>
*Screenshot: Mount type selection.*

Choose how files are mounted:

| Mount Type | Description |
|------------|-------------|
| **DFS** | Custom VFS with optimized streaming |
| **Rclone** | Embedded Rclone with full VFS features |
| **External Rclone** | Connect to existing Rclone RC instance |
| **None** | No mounting (API/WebDAV only) |

**Mount Path**: Where files appear (e.g., `/mnt/decypharr`)

**Cache Directory**: Local cache storage (e.g., `/cache`)

## Step 5: Review & Complete

Review settings and click **Complete Setup**.

You'll be redirected to the main dashboard with your API token displayed once.
